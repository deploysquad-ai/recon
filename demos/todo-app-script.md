# Demo Script: Todo App → recon graph

## User prompt (send to /recon)

> I'm building a personal todo app. It's for people who feel overwhelmed by their task lists.
> The core idea: you add tasks, group them into projects, and the app nudges you to focus on 
> one thing at a time. Mobile-first. There's a daily digest that emails you your top 3 priorities 
> for the day. Eventually we want a team mode where you can share projects with coworkers.

## What recon should infer

- Project: Todo App
- Goals: Reduce overwhelm, Focus on one task at a time, Daily email digest
- Personas: Overwhelmed Individual, Team Lead (future)
- Constraints: Mobile-first, Email delivery
- Modules: Task Manager, Project Groups, Daily Digest, Team Sharing (future/draft)
- User Stories: Add task, Group tasks into project, Get daily digest email, Share project
- Features: Task input, Project grouping, Priority nudge, Digest emailer, Team invite
- Version: v1 (solo, core flow), v2 (team mode)

## Confirmation response
yes

## Follow-up
generate context for the Priority Nudge feature
