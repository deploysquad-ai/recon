## Description

A minimal, fast-entry form for creating tasks. Optimized for one-tap access from the home screen and instant keyboard focus. Captures task name and optional project assignment.

## Scope

In scope: task name input, project selector, save/cancel. Out of scope: due dates, tags, attachments, sub-tasks (MVP).
