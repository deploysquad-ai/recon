## Description

A full-screen, distraction-free view that displays only the user's #1 priority task. Includes a 'I'm done' button and a 'Skip / blocked' option. Designed as the app's default home state.

## Scope

In scope: top task display, done + skip actions, project context label. Out of scope: timer/pomodoro, notes, sub-task checklist.
