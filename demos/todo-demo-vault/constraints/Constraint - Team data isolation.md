## Description

Users must only be able to see tasks and projects they own or have been explicitly invited to. Team members must not be able to access other teams' data. Personal tasks must remain private even for users who are also team members.

## Scope

Applies to the Team Collaboration module and any shared data queries in Task & Project Management.
