## Description

All UI must be designed and tested for mobile viewports first. Desktop is a secondary, enhanced experience. Touch targets, thumb zones, and single-column layouts are the baseline.

## Scope

Applies to all user-facing modules: Task & Project Management, Focus Engine, and Team Collaboration. The Daily Digest is email-rendered but must also be mobile-readable.
