## Story

As an Overwhelmed Individual, I want to be gently reminded to stay on my current task when I start to drift, so that I build a habit of finishing before switching.

## Rationale

Behavioral nudges are the mechanism that separates Focus from a plain todo app. The nudge must feel helpful, not punitive.
