## Story

As an Overwhelmed Individual, I want to quickly add a task to a project, so that I can capture what needs to be done without losing my flow.

## Rationale

Fast capture is critical for overwhelmed users. If adding a task takes too long, they abandon it and the system becomes incomplete.
